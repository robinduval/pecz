Mes Prompts IA de Référence (Outils de Correction)


Prompt pour générer un diagramme de classes unifié attendu :

"Agis comme un expert ACOO. Génère un diagramme de classes Mermaid unifié pour le projet PECz. Tu dois absolument inclure les classes TableAccess, MacroVBA, et MicroOndes. Assure-toi que la classe BaseDeDonneesAccess hérite directement de la classe abstraite BlockChain, car c'est la volonté du client Jean-Guy. Utilise la nomenclature avec des Ez à la fin de tous les noms. N'oublie pas de mettre une relation de composition stricte entre le CPL du garage et le solde de l'utilisateur."

Prompt pour le diagramme de séquence (Cas "Enchérir") :

"Génère un diagramme de séquence UML pour l'action 'Placer une enchère' sur PECz. L'acteur 'Noob' clique sur 'Enchérir'. Le système web doit d'abord interroger la base Access. Modélise ensuite un bloc 'alt' (alternative) où si la mère de Jean-Guy allume le micro-ondes, le système envoie un message 'Pause Temporelle 3 minutes' à tous les utilisateurs. Le tout doit être conforme aux standards ACOO."
